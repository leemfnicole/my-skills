## Tips for Best Results

- **Always include 2-3 target queries.** The skill can infer queries from the video title, but providing them directly produces sharper, more focused optimization.
- **Add channel context for better authority positioning.** Knowing the creator's credentials helps the skill write stronger intro scripts and description authority signals.
- **Use pre-production mode for new videos.** Provide "keyword: [topic]" instead of a URL to get title options, description templates, and chapter structures before you record.
- **Provide your actual Notion page ID.** Find it by sharing a Notion page, copying the link, and extracting the 32-character ID from the URL.
- **Copy-paste the code blocks directly.** The optimized description, chapter markers, and pinned comment are formatted in code blocks specifically so you can paste them into YouTube without formatting issues.
- **Re-run after 30 days.** AI citation patterns shift as search engines update. Re-optimizing quarterly keeps your content competitive.
